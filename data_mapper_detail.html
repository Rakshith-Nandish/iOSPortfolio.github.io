Data Mapper Inflight

The application lets users control and intelligently fly DJI drones using smartphones or tablets

Project summary

PrecisionFlight mobile app turns a DJI drone into an advanced remote sensing a tool that empowers businesses and consumers to gain actionable aerial data autonomously. Users can obtain areial images for analysis, post analysis users can apply automated analysis of their 2D/3D maps using DataMappers's marketplace. Users can easily select and edit the area of interest using the application, the application then calculates the optimum flight path and takes images at predecided intervals.
